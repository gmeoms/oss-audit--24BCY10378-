## 📌 Project Overview
This project is an extensive audit of Git as a Free and Open Source Software (FOSS). It covers the historical shift from proprietary systems (BitKeeper) to decentralized version control, examines the **GPL v2** licensing model, and provides a suite of Bash shell scripts for system auditing, package management, and log analysis.

## 💻 Environment Requirements
To run the scripts in this repository, you need:
* **Operating System:** Linux (Optimized for **Ubuntu 24.04 LTS** or WSL2).
* **Shell:** Bash[cite: 218].
* **Package Manager:** `dpkg` and `apt`.
* **Dependencies:** Git and standard GNU coreutils.

---

## 🚀 Setup & Installation

### 1. Update Your System
Ensure your local package index is up to date:
```bash
sudo apt update && sudo apt upgrade -y
```

### 2. Install Git
If Git is not already present, install it using the following command:
```bash
sudo apt install git -y
```

### 3. Verify Installation
Confirm that Git is installed and locate its binary:
```bash
git --version
which git
```

## 🛠️ Execution Instructions

The project includes five primary scripts. Follow these steps to set permissions and execute them.

### Step 1: Make Scripts Executable
Run this command in the directory containing the scripts:
```bash
chmod +x script1.sh script2.sh script3.sh script4.sh script5.sh
```

### Step 2: Running the Scripts

| Script | Description | Execution Command |
| :--- | :--- | :--- |
| **Script 1** | **System Identity Report:** Displays Kernel, Distro, User, and Uptime. | `./script1.sh` |
| **Script 2** | **FOSS Package Inspector:** Audits the Git package status and version. | `./script2.sh` |
| **Script 3** | **Directory Auditor:** Checks permissions and sizes of system directories. | `./script3.sh` |
| **Script 4** | **Log File Analyzer:** Searches a log file for a specific keyword. | `./script4.sh <log_path> <keyword>` |
| **Script 5** | **Manifesto Generator:** Creates a personalized `.txt` manifesto file. | `./script5.sh` |

> **Note for Script 4:** If you do not provide arguments, the script will prompt you to enter a valid log file path.
